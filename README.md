# _2.프로젝트 소개_

#### MARCAT 프로젝트는 중앙 관리가 필요 없는 중고 거래 시스템입니다.

#### MARCAT은 사용자간의 쪽지 전달과 거래 요청으로 거래를 진행할 수 있으며,

#### 동네 커뮤니티 기능으로 소셜 네트워크를 구축할 수 있습니다.

# _5.프로젝트 리뷰_

### 보완하면 좋을 부분
#### - 프로젝트 이름에 걸맞게 고양이에 대한 상품에 대한 이벤트나 주력 상품을 배치 

### 아쉬운 점
#### - 코딩 스타일 일관성을 최대한 추려하였지만 쉽지 않았음.
#### - 초기 계획에서 여러 이슈나 팀원과의 합의된 판단으로 중간 중간 기획이 다급하게 바뀜.

### 시도하면 좋을 부분
#### - 개발 환경을 SpringBoot + jpa로 바꿔서 적용하는 부분
